#### centos7 - php73

```shell
# https://tecadmin.net/install-php7-on-centos7/
sudo yum localinstall -y http://rpms.famillecollet.com/enterprise/remi-release-7.rpm
sudo yum --enablerepo=remi-php73 install -y php php-fpm \
        php-mysqlnd php-pecl-redis php-pecl-memcached
sudo systemctl enable php-fpm
php -m |grep -E '(mysql|redis|memcache)'
```

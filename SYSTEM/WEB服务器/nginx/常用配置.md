---

http 请求中的客户端 IP

```
if ($remote_addr != 192.168.1.1) {
    return 403;
}
```

---

http 请求中的主机名称、或与请求匹配的服务器名称

```
if ($host != dom.com) {
    return 403;
}
```

---

http 请求中的 cookie

```
if ($http_cookie != 'id=admin') {
    return 403;
}
```

---

请求行中的参数

```
$args
```

用户请求的地址，常用来做跳转参数

```
return 301 https://$host/$request_uri;
```

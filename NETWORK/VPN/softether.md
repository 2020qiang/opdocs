## SoftEther VPN

<https://www.softether.org>



### vpncmd 常用命令

网卡

```
创建 NicCreate
删除 NicDelete
查看 NicList
```

连接

```
创建 AccountCreate
密码 AccountPasswordSet
删除 AccountDelete
自连 AccountStartupSet
连接 AccountConnect
断开 AccountDisconnect
查看 AccountList
```
